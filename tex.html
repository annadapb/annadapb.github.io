Styles in TikZ/PGF

To make diagrams uniform and make it easier to change later, styles can be added like below. Consider the following TikZ snippet, 

\begin{tikzpicture}
\path
	(3, 0) node[draw, circle, inner sep=3pt]{}
	(2, 1) node[draw, circle, inner sep=3pt]{}
	(1, 2) node[draw, circle, inner sep=3pt]{}
	(0, 3) node[draw, circle, inner sep=3pt]{} ;
\end{tikzpicture}
Now if want to change the circle to a rectangle, we have to change every instance of the circle. By defining styles, we can do all the changes at once by changing the definition. 

\begin{tikzpicture}
[style name/.style = { draw, circle, inner sep=3pt,}, ]
\path
	(3, 0) node[style name]{}
	(2, 1) node[style name]{}
	(1, 2) node[style name]{}
	(0, 3) node[style name]{} ;
\end{tikzpicture}
And now all we need to do is change to, [style name/.style = { draw, rectangle, inner sep=3pt,}, ]
